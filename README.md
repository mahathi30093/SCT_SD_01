# 🌡️ Temperature Converter

## 📌 SkillCraft Technology Internship - Task 01

This project is a simple Python-based Temperature Converter that allows users to convert temperatures between Celsius, Fahrenheit, and Kelvin scales.

## 🚀 Features

- Celsius to Fahrenheit
- Celsius to Kelvin
- Fahrenheit to Celsius
- Fahrenheit to Kelvin
- Kelvin to Celsius
- Kelvin to Fahrenheit
- User-friendly menu-driven interface

## 🛠️ Technologies Used

- Python 3
- Visual Studio Code
- Git & GitHub
